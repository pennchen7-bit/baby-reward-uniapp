## 📝 使用说明

### 立即开始开发

```bash
# 进入项目目录
cd baby-reward-uniapp

# 安装依赖
npm install

# 开发模式（微信小程序）
npm run dev:mp-weixin

# 开发模式（H5 - 浏览器测试）
npm run dev:h5
```

### 用微信开发者工具打开

1. 运行 `npm run dev:mp-weixin`
2. 打开微信开发者工具
3. 导入项目，选择目录：`dist/dev/mp-weixin`
4. 在工具中预览
