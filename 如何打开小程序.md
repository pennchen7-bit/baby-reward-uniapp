## ⚠️ 常见问题

### 1. 找不到微信开发者工具
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 下载「macOS」版本
- 下载完成后拖到「应用程序」文件夹

### 2. AppID 不匹配
- 确保导入的是 `dist/dev/mp-weixin` 目录
- project.config.json 中已配置正确的 AppID

### 3. 编译错误
- 检查 Node.js 版本（推荐 18.x 或 20.x）
- 重新运行 `npm install`
- 重新编译：`npm run dev:mp-weixin`
